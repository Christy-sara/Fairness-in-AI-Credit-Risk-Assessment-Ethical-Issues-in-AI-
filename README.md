A machine learning fairness project using the German Credit Dataset to predict credit risk, evaluate gender-based bias, and apply reweighing to improve fairness in credit decision-making.

Dataset: German Credit Dataset. (https://www.kaggle.com/datasets/jumpingdino/german-credit-dataset)

Models: Logistic Regression, Decision Tree, Random Forest.

Fairness Focus: Gender as the protected attribute.

Mitigation Method: Reweighing.

Tools: Python, pandas, scikit-learn, matplotlib, mlxtend.
